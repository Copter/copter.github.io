Escape Plan

 https://github.com/Copter/2190472-netcentric-escape-room-frontend, https://github.com/Seqlet/2190472-netcentric-escape-room-backend

 2020

 Netcentric Architecture Project

 React Frontend, Art

 Norapat buppodom, Ratchanon Eksombatchai, Summa Sriworapong

Escape plan is a simple, online multiplayer game created as the final project for the Netcentric Architecture course. The main focus of this project is to be a proof-of-concept for creating a system that users can create and join a room using the generated code, and able to interact with each other within the room.
