Aincrad

 https://github.com/Ukkabanana/aincrad

 2019

 Advanced Computer Programming Project

 Ionic, Angular, Firebase

 Arin Thirasawasd

Aincrad is an application for board game enthusiast using Ionic framework and Angular. The user can create a digital library of board games by listing games they have already played, bookmarking games they want to play in the future, and leaving a review for the games they played. The application is powered by Board Game Geek API and used Firebase as user database.
